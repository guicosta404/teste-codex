# Jogo da Cobrinha

Projeto simples do famoso jogo da cobrinha feito com HTML, CSS e JavaScript puro, sem dependencias externas.

## Como jogar

- Controle a cobra com as setas do teclado ou `W`, `A`, `S`, `D`.
- Cada fruta coletada aumenta a cobrinha em 1 bloco.
- A dificuldade sobe gradualmente conforme a pontuacao aumenta.
- O jogo termina ao bater na parede ou no proprio corpo.

## Como executar

1. Abra o arquivo `index.html` no navegador.
2. Jogue diretamente na pagina.
3. Quando perder, reinicie com `Enter` ou pelo botao na interface.

## Estrutura

- `index.html`: estrutura da pagina e elementos da interface
- `style.css`: estilos visuais e layout responsivo
- `script.js`: logica do jogo, controles, pontuacao e aumento de dificuldade
